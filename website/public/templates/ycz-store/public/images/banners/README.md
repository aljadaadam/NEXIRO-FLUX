# Banner images

ضع صور البنرات هنا.

- المسار العام في الموقع سيكون: `/images/banners/<filename>`
- صيغ مقترحة: `.png` أو `.jpg` أو `.webp`
- أسماء مقترحة للبدء:
  - `home-hero.png`
  - `services-banner.png`

ملاحظة: نظام الملفات في لينكس حساس لحالة الأحرف (Case-sensitive)، لذا تأكد أن الاسم في الكود يطابق اسم الملف تمامًا.
